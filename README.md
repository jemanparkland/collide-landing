# COLLIDE - Creator x Brand Marketplace

A landing page for connecting brands with content creators.

## Quick Start

1. Open `index.html` in a web browser
2. Or deploy to any static hosting service (Render, Netlify, Vercel, etc.)

## Features

- Responsive design
- Separate signup forms for brands and creators (via Tally)
- Animated elements and micro-interactions
- Mobile-friendly

## Deployment to Render

1. Push this repository to GitHub
2. Connect your GitHub repo to Render
3. Choose "Static Site" as the service type
4. Deploy!

## Tech Stack

- Pure HTML/CSS/JavaScript
- Tally forms for data collection
- No build process required
